# 📝 To-Do List Application

A responsive and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.  
This project was developed as an internship project to practice frontend development concepts like DOM manipulation, Local Storage, and responsive UI design.

---

## 📌 Project Overview

The To-Do List Application allows users to:

- Create new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Store tasks permanently using browser Local Storage
- Switch between light and dark mode

The application works smoothly on desktop and mobile devices with a responsive design.
